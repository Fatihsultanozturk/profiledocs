Touch Bar (Control Strip)¶

Domain
com.apple.controlstrip

Keys¶

MiniCustomized¶

This key can be used to customise the touch bar layout.

It is an array containing bundle IDs for each button that should be available.

Choices:

  • com.apple.system.do-not-disturb
  • com.apple.system.media-play-pause
  • com.apple.system.sleep
  • com.apple.system.screen-lock
