Per-App VPN¶

The Per-App VPN payload is used for configuring add-on VPN software, and it works only on VPN services of type ‘VPN’. It should not be confused with the standard VPN payload, described in VPN.

The Per-App VPN payload supports all of the keys described in VPN plus the following additional keys

Note

Copy VPN payload keys into the per app vpn manifest.
