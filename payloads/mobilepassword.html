maxFailedAttempts integer Maximum Number of Failed Attempts Number of passcode entry attempts allowed before all data on device will be erased n/a

maxPINAgeInDays integer Maximum Passcode Age The number of days for which the passcode can remain unchanged n/a

minLength integer Minimum Passcode Length Smallest allowable number of characters in passcode n/a
requireAlphanumeric boolean Require Alphabetic Value Require passcodes to contain at least one letter n/a
pinHistory integer Passcode History Number of unique passcodes required between passcode reuse n/a
