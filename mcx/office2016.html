Microsoft Office 2016¶

These are custom settings and must be placed in a com.apple.ManagedClient.preferences payload.

Summary¶

Name Type Title Description Required
kCUIThemePreferencesThemeKeyPath integer Colorful Theme Sets Office theme: 0 - Colorful, 1 - Classic n/a
OfficeActivationEmailAddress string Set Belongs to field Sets value of ‘Belongs to’ field in the About Box. This happens automatically on 15.33+ installs n/a
DefaultsToLocalOpenSave boolean Save/Open defaults to ‘On my Mac’ When set to true will force the open/save panel to ‘On my Mac’ instead of ‘Online Locations’ n/a
ShowWhatsNewOnLaunch boolean Show ‘Whats New’ dialog on launch Replaces the ‘OUIWhatsNewShownItemIds’ key as a single place to remove all “What’s new” messages for all suite apps: Word, PowerPoint, Excel, Outlook, and OneNote. In build 170420 and later. n/a
VisualBasicMacroExecutionState string VisualBasic Macro security level Sets the VisualBasic Macro security level. Valid values: DisabledWithoutWarnings, DisabledWithWarnings, EnabledWithWarnings n/a
msoridEnableLogging integer authentication logging Set to 1 to increase logging levels for authentication requests n/a
msoridDefaultMinimumSeverity integer msoridDefaultMinimumSeverity Set value to 200 if msoridEnableLogging is enabled n/a
