Microsoft OneDrive¶

These are custom settings and must be placed in a com.apple.ManagedClient.preferences payload.

Summary¶

Name Type Title Description Required
DisablePersonalSync boolean Disable personal OneDrive accounts Blocks users from signing in and syncing files in personal OneDrive accounts. If this key is set after a user has set up sync with a personal account, the user will be signed out. n/a
AutomaticUploadBandwidthPercentage integer Percentage of local bandwidth used This parameter determines the percentage of local upload bandwidth that the sync client can use. Accepted values are from 1 through 99. n/a
HideDockIcon boolean Hide the OneDrive dock icon when running When set to true, this parameter hides the OneDrive dock icon even when the application is running. n/a
IsBusinessProvisioned boolean IsBusinessProvisioned n/a
DefaultToBusinessFRE boolean Default to OneDrive Business Default to OneDrive Business instead of OneDrive Personal n/a
EnableAddAccounts boolean Allow users to add accounts to OneDrive Allow users to add accounts to OneDrive n/a
Tenants dict Specify default location by tenant Specifies the default location of the OneDrive folder for each tenant n/a
